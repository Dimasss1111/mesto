1. Содержание:

  * Имя ученого
  * Его заслуги
  * Исследованные места

  * submit

3. Ссылка на Git Pages:
  https://dimasss1111.github.io/mesto/

 * Добавление новых мест
 * Постановка лайков

 * Валидация форм
 * Закрытие всплывающих окон нажатием на Esc или оверлей
